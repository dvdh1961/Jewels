Dear Retro Fans,

This game is my first in 43 years since the day I bought my ColecoVision / Coleco Adam computer.
I still own it today, together with many options and expansions I collected over the years.

My love for this hardware runs so deep that I even wrote my own emulator and I am currently working on a hardware project to bring the Adam back to life.

In the meantime, I kept myself busy with the wonderful program created by Oscar Toledo Gutiérrez — his CVBasic system allows us to create ROMs in a very structured way. A fantastic tool!

This game is still in an early stage. If anyone would like to contribute graphics or music, it would be greatly appreciated. Of course, contributors will be credited in the game titles.

This is an open-source project, and you are free to use the software as you wish to learn from it.

Have fun with it! If you like this little project, feel free to give it a star ⭐

Thanks,
Danny 🎮
